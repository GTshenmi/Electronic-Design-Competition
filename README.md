# Electronic-Design-Competition

###  `放大器非线性失真研究装置`

#### `项目介绍`
-------

        本项目是根据2020江苏省电子设计竞赛E题-放大器非线性失真研究装置的要求，参考集成运放电路的设计原则，制作出了一款放大器非线性失真
    研究装置，能够生成放大后的标准正弦波，以及四种失真的正弦波形（顶部失真、底部失真、双向失真、交越失真波形），并基于stm32f407平台，计
    算出任意波形的总谐波失真，显示在OLED屏幕上。
